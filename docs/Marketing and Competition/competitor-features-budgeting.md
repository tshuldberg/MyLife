# Competitor Features: Budgeting Apps

Comprehensive feature analysis of the top budgeting applications competing in the personal finance space.

---

## 1. YNAB (You Need A Budget)

**Market Position:** ~700K MAU, $49M+ ARR. Zero-based/envelope budgeting leader.
**Pricing:** $14.99/month or $109/year. No free tier. 34-day free trial. Free for college students for one year.
**Platforms:** Web, iOS, Android, iPad, Apple Watch, Alexa.

### Budget Management

| Feature | Description |
|---------|-------------|
| Zero-Based Budgeting | Every dollar you currently have is assigned a specific job (spending, saving, debt). You only budget money you actually possess, not projected income. |
| Four Core Rules | Methodology: (1) Give Every Dollar a Job, (2) Embrace Your True Expenses, (3) Roll With the Punches, (4) Age Your Money. |
| YNAB Targets | Set spending and savings targets with flexible cadences: weekly, monthly, yearly, or custom timeframes. |
| Category Templates | Pre-built category structures you can import and customize to jumpstart budgeting for life events (new baby, moving, weddings, etc.). |
| Focused Views | Customize which budget categories are visible, hiding irrelevant ones to reduce visual noise. |
| True Expenses | Plan for irregular but predictable expenses (annual insurance, car repairs, holiday gifts) by saving monthly. |
| Roll With the Punches | Move money between categories when overspending occurs instead of treating the budget as broken. |
| Spending Breakdown | Visual breakdown of spending by category with drill-down into individual transactions. |

### Bank Linking & Account Sync

| Feature | Description |
|---------|-------------|
| Automatic Bank Import | Securely link accounts through Plaid, MX, and TrueLayer data providers for automatic transaction import. |
| Apple Card/Cash/Savings Import | Native integration for importing Apple Card, Apple Cash, and Savings with Apple Card transactions. |
| File-Based Import | Drag-and-drop import of transaction files in CSV, QFX, OFX, and QIF formats downloaded from banks. |
| Manual Transaction Entry | Enter transactions by hand as they happen for real-time budget awareness. |
| Multi-Account Support | Link checking accounts, savings accounts, credit cards, and loans. |

### Transaction Management

| Feature | Description |
|---------|-------------|
| Transaction Matching | Automatically matches manually entered transactions with imported bank transactions. |
| Transaction Categorization | Assign transactions to budget categories with auto-categorization learning. |
| Transaction Search & Filter | Search and filter transactions across all accounts by payee, category, amount, date, etc. |
| Memo & Flag System | Add memos to transactions and flag them with colors for follow-up. |
| Split Transactions | Split a single transaction across multiple budget categories. |
| Scheduled Transactions | Set up recurring transactions (rent, subscriptions) that auto-enter on their due date. |
| Reconciliation | Reconcile accounts against bank statements to catch discrepancies. |

### Goals & Savings

| Feature | Description |
|---------|-------------|
| Savings Targets | Set target amounts for categories with visual progress bars. |
| Monthly Contribution Targets | Define how much to save per month toward a specific goal. |
| Target Date Goals | Set a date by which you want to reach a savings target; YNAB calculates monthly contributions needed. |
| Custom Target Types | Choose between "needed for spending" (bills), "savings balance" (emergency fund), and "monthly savings builder." |

### Reports & Analytics

| Feature | Description |
|---------|-------------|
| Spending Report | Bar/line charts showing spending by category over time, filterable by date range and category. |
| Net Worth Report | Track net worth trajectory over time across all linked accounts. |
| Income vs. Expense Report | Compare income to spending month-over-month. |
| Age of Money | Proprietary metric showing how many days pass between earning and spending, indicating financial health (30+ days = living on last month's income). |
| Spotlight Dashboard | Shows what needs attention now: overspent categories, upcoming targets, budget health. |

### Debt Management

| Feature | Description |
|---------|-------------|
| Credit Card Management | Track credit card spending against budget categories; payment category auto-calculates based on budgeted spending. |
| Loan Planner | Simulator showing how extra payments reduce interest and payoff time. Calculate interest saved per extra dollar applied. |
| Debt Category Tracking | Link credit cards to budget; as you budget for spending on the card, the payment amount builds automatically. |

### Collaboration

| Feature | Description |
|---------|-------------|
| YNAB Together | Share one subscription with up to 5 additional users (6 total). Partners, parents, or caregivers can all access the same budget. |
| Multi-Budget Support | Create separate budgets (personal, business, household) under one account. |

### Notifications & Alerts

| Feature | Description |
|---------|-------------|
| Overspending Alerts | Notifications when a category is overspent. |
| Transaction Import Alerts | Notifications when new transactions are imported from linked accounts. |
| Target Alerts | Alerts when you're behind on funding a target. |

### Education & Support

| Feature | Description |
|---------|-------------|
| Live Workshops | Free live classes teaching budgeting methodology and app usage. |
| Video Tutorials | Library of instructional videos. |
| Help Center | Comprehensive knowledge base. |
| Customer Support | Email and chat support. |

### Import/Export & API

| Feature | Description |
|---------|-------------|
| File Import | CSV, QFX, OFX, QIF file import from banks. |
| YNAB API | Public REST API for developers to build integrations and access budget data programmatically. |
| Toolkit for YNAB | Community-built browser extension adding power-user features (Days of Buffering, running balance, pacing, etc.). |

### Platform Features

| Feature | Description |
|---------|-------------|
| Multi-Device Sync | Real-time sync across web, mobile, tablet, and watch. Works offline with sync on reconnect. |
| Mobile Widgets | Home screen widgets showing budget category balances. |
| Apple Watch App | Quick-glance budget status and transaction entry from the wrist. |
| Alexa Integration | Voice commands for checking category balances. |
| Ad-Free Experience | No ads, no third-party products promoted within the app. |

## 2. Monarch Money

**Market Position:** $850M valuation, fastest growing post-Mint shutdown. Comprehensive budgeting + wealth tracking.
**Pricing:** $14.99/month or $99.99/year (~$8.33/month). No free tier. 7-day free trial.
**Platforms:** Web, iOS, Android.

### Budget Management

| Feature | Description |
|---------|-------------|
| Category Budgeting | Traditional approach: set spending limits for individual categories (groceries, dining, entertainment, etc.). |
| Flex Budgeting | Alternative approach: groups spending into fixed expenses, flexible expenses, and non-monthly costs for a less rigid budgeting experience. |
| Custom Categories | Create, rename, and organize categories and category groups to match your financial structure. |
| Recommended Categories | Smart suggestions for recategorizing transactions based on common patterns. |
| Rollover Budgets | Unused budget amounts can roll over to the next month. |
| Budget Notifications | Alerts when approaching or exceeding category limits. |

### Bank Linking & Account Sync

| Feature | Description |
|---------|-------------|
| Multi-Provider Sync | Connections to 13,000+ financial institutions through multiple data providers. |
| Account Types | Link checking, savings, credit cards, retirement accounts, brokerage accounts, and loans. |
| Connectivity Dashboard | Transparency into account connection health metrics, updated daily. Shows sync status and issues. |
| Manual Accounts | Create manual accounts for tracking assets or liabilities not available through automatic sync. |

### Transaction Management

| Feature | Description |
|---------|-------------|
| AI Smart Categorization | Machine learning automatically categorizes transactions based on past behavior, improving over time. |
| Merchant Rules | Set rules to automatically categorize transactions from specific merchants, by amount, or by name. |
| Recommended Merchants | Smart suggestions for merchant assignment based on common patterns. |
| Transaction Tags | Assign colored tags (tax, reimburse, split, business, subscription) to transactions for cross-category organization. Default and custom tags supported. |
| Transaction Notes | Add notes and context to individual transactions. |
| Split Transactions | Divide a single transaction across multiple categories. |
| Transaction Search | Search and filter by date, category, merchant, account, tag, amount, and more. |
| Receipt Scanning | Upload a photo of a receipt; Monarch auto-attaches it to the correct transaction, adds purchase details, and can split one generic transaction into properly categorized items. |
| Transaction Assignment | Assign transactions to household members for review. |

### Goals & Savings

| Feature | Description |
|---------|-------------|
| Save Up Goals | Dedicated goal type for saving toward a target (emergency fund, vacation, home down payment). Visual progress tracking. |
| Debt Pay Down Goals | Separate goal type designed for debt reduction with different planning logic than savings goals. |
| Goal Forecasting | Projects when you'll reach your goal based on current contribution pace. |
| Scenario Planning | Simulate how achieving one goal affects others in your financial plan. |

### Reports & Analytics

| Feature | Description |
|---------|-------------|
| Spending Reports | Detailed spending breakdowns by category, merchant, tag, or account over custom time periods. |
| Income Reports | Track income sources and trends over time. |
| Cash Flow Reports | Visualize inflows and outflows with bar charts and trend analysis. |
| Sankey Diagram | Visual flow chart showing how money moves from income sources through spending categories. Can be shared with amounts hidden. |
| Net Worth Tracking | Automated net worth calculation across all synced accounts with historical trend visualization. |
| Custom Report Filters | Filter any report by category, merchant, account, tag, amount, and dynamic timeframes. |
| Saved Reports | Save favorite report configurations for quick access. |
| Report Download | Export report charts for sharing or further analysis. |
| Forward Cash Flow Projection | Projects future account balances based on upcoming income and recurring expenses. |

### Investment Tracking

| Feature | Description |
|---------|-------------|
| Portfolio Tracking | View investment performance across all linked brokerage and retirement accounts. |
| Asset Allocation View | Monitor how investments are distributed across asset classes. |
| Gains & Losses | Track investment gains and losses over time. |
| Equity Compensation Tracking | Complete view of stock options, RSUs, and other equity comp. |
| Net Worth Integration | Investment accounts feed into overall net worth tracking. |

### Subscription & Recurring Management

| Feature | Description |
|---------|-------------|
| Recurring Detection | Auto-scans accounts to identify recurring charges (subscriptions, memberships, bills). |
| Recurring Calendar | Calendar view showing when bills are due and recurring charges expected. |
| Manual Recurring | Manually add recurring expenses that aren't auto-detected. |
| Subscription Monitoring | Track app memberships, gym fees, streaming services, and other recurring subscriptions. |

### Credit & Debt

| Feature | Description |
|---------|-------------|
| Credit Score Tracking | Monitor credit score with custom notifications for changes. |
| Credit Health Tracking | Method introduced with Target extension for tracking credit health. |
| Debt Paydown Goals | Structured plans for paying off debt with progress visualization. |

### Collaboration

| Feature | Description |
|---------|-------------|
| Shared Household Access | Invite a partner to manage finances jointly at no extra cost. Real-time collaboration. |
| Shared Views | "Yours, mine, and ours" labels that bring clarity to shared finances. Customizable permissions. |
| Financial Advisor Access | Invite a financial advisor to view your data at no additional cost. |
| Monarch for Professionals | Program for financial professionals to collaborate with all clients through a portal. |
| Transaction Comments | Add comments to transactions visible to all household members. |

### AI Features

| Feature | Description |
|---------|-------------|
| AI Assistant | Financial expert that knows your complete financial picture and can answer any question in natural language. |
| Smart Categorization | AI-powered transaction categorization that learns from corrections. |
| AI Insights | Real-time suggestions and alerts about spending patterns and anomalies. |

### Extensions & Integrations

| Feature | Description |
|---------|-------------|
| Amazon Order Extension | Syncs with Amazon orders to accurately categorize and split Amazon transactions automatically. |
| Target Extension | Supports Target shopping trip categorization and credit health tracking. |

### Import/Export

| Feature | Description |
|---------|-------------|
| CSV Import | Import transaction history via CSV with automatic column mapping (category, tags, notes auto-recognized). |
| Balance History Import | Import historical balance data for accounts. |
| CSV Export | Download all transactions across all accounts in CSV format. |
| Mint Import | Migrated Mint users can import their full history. |

### Notifications

| Feature | Description |
|---------|-------------|
| Budget Alerts | Notifications when approaching or exceeding budget limits. |
| Bill Reminders | Alerts for upcoming recurring payments. |
| Credit Score Changes | Custom notifications when credit score changes. |
| Account Connection Issues | Alerts when a bank connection needs attention. |

### Security

| Feature | Description |
|---------|-------------|
| SOC 2 Compliance | Independently audited security controls. |
| Bank-Level Encryption | Industry-standard data protection. |
| Read-Only Access | Bank connections are read-only; Monarch cannot move money. |
