## 1. Notion

**Category:** All-in-one workspace (notes, docs, databases, projects, wikis, AI agents)
**Valuation:** $11B+ | **Users:** 100M+ | **ARR:** ~$600M
**Owner:** Notion Labs, Inc. (independent, venture-backed)
**Platforms:** Web, macOS, Windows, iOS, Android
**Pricing Model:** Freemium per-seat SaaS

### Editing & Content Creation

- **Block-based editor** -- every piece of content (text, image, embed, database) is a modular block that can be nested, rearranged, and transformed
- **50+ block types** including text, headings (H1/H2/H3), to-do lists, bulleted lists, numbered lists, toggle lists, quotes, dividers, callouts, code blocks, tables, embeds, images, videos, audio, files, bookmarks, equations (KaTeX), table of contents, breadcrumbs, and synced blocks
- **Markdown support** -- full inline markdown shortcuts for bold, italic, strikethrough, code, headings, lists, links, and block quotes
- **Slash commands** -- type `/` to insert any block type or run actions from the keyboard
- **Drag-and-drop** -- reorder and nest any block via drag handles
- **Multi-column layouts** -- arrange blocks side by side in column layouts
- **Synced blocks** -- create blocks that mirror across multiple pages; editing one updates all instances automatically
- **Code blocks** -- syntax highlighting for 60+ languages with line numbers and one-click copy
- **Toggle blocks** -- collapsible content sections for FAQs, documentation, and nested organization
- **Callout blocks** -- highlighted boxes with icons for tips, warnings, and notes
- **Embeds** -- inline embeds for 500+ services (Google Maps, Figma, Loom, Miro, GitHub Gists, Tweets, Spotify, and more)
- **Page nesting** -- infinite hierarchical nesting of pages within pages
- **Inline databases** -- embed database views directly inside pages alongside other content
- **Equation blocks** -- LaTeX/KaTeX math rendering for academic and technical content
- **Web Clipper** -- browser extension to save web pages, articles, and selected content directly to Notion

### Databases & Views

- **6+ database views** -- Table, Board (Kanban), Calendar, Timeline (Gantt), List, Gallery, and Chart views; each database can have unlimited views
- **Chart view** -- bar, line, and donut chart visualizations of database data for reporting and analysis
- **Property types** -- text, number, select, multi-select, date, person, files/media, checkbox, URL, email, phone, formula, relation, rollup, created time, created by, last edited time, last edited by, status, and unique ID
- **Filters** -- filter database entries by any property; supports AND/OR logic with filter groups nested up to 3 layers deep
- **Sorts** -- sort by any property, ascending or descending, with multi-level sort chains
- **Grouping** -- group database items by select, multi-select, person, date, or other properties
- **Relations** -- link entries across databases to create relational data models
- **Rollups** -- pull and calculate data from related entries (sum, average, count, min, max, show original, etc.)
- **Formulas** -- write custom formulas using 40+ built-in functions (if/then, concat, date math, regex, etc.)
- **Database templates** -- define reusable entry templates for consistent data entry
- **Conditional coloring** -- color-code database cells based on values, formulas, relations, and rollups
- **Linked databases** -- reference the same database in multiple locations with independent views and filters
- **Sub-items** -- parent-child relationships within a single database for hierarchical project views

### Forms

- **Notion Forms** -- create native forms linked to any database for internal or external data collection
- **Conditional logic** -- show/hide form fields based on previous answers (Business and Enterprise plans)
- **Public forms** -- shareable form links for external respondents without Notion accounts
- **Form responses** -- automatically populate database entries from form submissions

### AI Features

- **Notion AI** -- built-in AI assistant for writing, summarizing, translating, improving tone, explaining, and brainstorming
- **AI writing tools** -- generate drafts, improve writing, fix grammar/spelling, make text shorter/longer, change tone, translate to 14+ languages
- **AI summarization** -- summarize pages, databases, and meeting notes with one click
- **AI Q&A** -- ask questions about workspace content and get answers sourced from your pages and databases
- **Custom AI Agents** -- autonomous AI workers that handle recurring tasks 24/7 across Notion, Slack, Mail, Calendar, Figma, Linear, and custom MCP servers
- **Agent triggers** -- schedule agents on timers or trigger them from events (new database entry, Slack message, email received)
- **Multi-model support** -- Agents can use GPT-5.2, Claude Opus 4.5, Gemini 3, and other models
- **AI Meeting Notes** -- automatic transcription, summaries, and action items from meetings (mobile and desktop)
- **Enterprise AI Search** -- search across Notion, Slack, Google Drive, Confluence, and other connected apps from one search bar
- **AI autofill** -- automatically populate database properties using AI based on page content

### Collaboration

- **Real-time co-editing** -- multiple users can edit the same page simultaneously with live cursors
- **Comments** -- inline and page-level comments with @mentions and threads
- **@mentions** -- tag people, pages, dates, and databases inline
- **Guest access** -- invite external collaborators to specific pages without full workspace membership
- **Teamspaces** -- organize workspace into team-level containers with independent settings and permissions
- **Private teamspaces** -- restrict teamspace visibility and access (Business and Enterprise)
- **Permissions** -- granular permission levels: full access, can edit, can comment, can view; apply at workspace, teamspace, page, or database level
- **Page locking** -- lock pages to prevent accidental edits
- **Page verification** -- mark pages as verified/approved (Business plan)
- **People directory** -- built-in directory showing all workspace members with customizable profile fields; import from HR systems

### Organization & Navigation

- **Sidebar** -- hierarchical page tree with favorites, private pages, teamspaces, and shared pages
- **Search** -- full-text search across all pages, databases, and comments; filter by date, creator, teamspace
- **Favorites** -- pin frequently used pages for quick access
- **Breadcrumbs** -- navigate page hierarchy via breadcrumb trail
- **Page history** -- version history with rollback: 7 days (Free), 30 days (Plus), 90 days (Business), unlimited (Enterprise)
- **Trash** -- deleted pages go to trash with 30-day recovery window
- **Import** -- import from Evernote, Confluence, Asana, Trello, Google Docs, Dropbox Paper, Word, HTML, CSV, and Markdown
- **Export** -- export pages as Markdown, CSV, PDF, or HTML

### Templates

- **Template gallery** -- 500+ community and official templates across categories (project management, CRM, wiki, personal, design, engineering, HR, etc.)
- **Custom templates** -- create reusable page and database entry templates
- **Template buttons** -- one-click buttons that generate pre-configured page structures

### Calendar & Scheduling

- **Notion Calendar** -- standalone calendar app (web, macOS, iOS) with two-way database sync
- **Database-calendar sync** -- database date properties appear as calendar events and vice versa
- **Scheduling links** -- generate shareable booking links with multi-invitee support, conflict checking, and rescheduling
- **Google Calendar integration** -- two-way sync with Google Calendar

### Mail

- **Notion Mail** -- AI-powered email client for Gmail users; save emails to Notion pages, AI-assisted triage and drafting

### Integrations & API

- **REST API** -- full CRUD API for pages, databases, blocks, users, comments, and search
- **Webhooks** -- real-time event notifications for page and database changes to trigger external workflows
- **Native integrations** -- Slack, Google Drive, GitHub, Jira, Figma, Asana, Trello, Zoom, Loom, and more
- **Zapier/Make** -- 3,000+ automation connections via Zapier and Make
- **MCP server integrations** -- Notion as an MCP data source for AI tools (Lovable, Perplexity, Mistral, HubSpot)
- **Embed support** -- 500+ embeddable services via iframe and oEmbed
- **Import/export ecosystem** -- CSV, Markdown, HTML, PDF import and export

### Security & Administration

- **SAML SSO** -- single sign-on via Okta, Azure AD, OneLogin, etc. (Business and Enterprise)
- **SCIM provisioning** -- automatic user provisioning and deprovisioning (Enterprise)
- **Audit log** -- detailed activity logs for compliance and security review (Enterprise)
- **Domain management** -- claim and manage email domains for automatic workspace membership
- **Content export** -- workspace-wide data export for backup and compliance
- **SOC 2 Type II** -- certified security compliance
- **GDPR compliant** -- EU data protection compliance
- **2FA** -- two-factor authentication for all users

### Notion Sites

- **Public websites** -- publish Notion pages as public-facing websites with custom domains
- **Custom domains** -- use your own domain for published Notion Sites
- **SEO controls** -- meta titles, descriptions, and social preview images
- **Google Analytics** -- embed analytics tracking on published sites

### Platform & Performance

- **Offline mode** -- basic offline editing with sync on reconnection (added August 2025; limited block type support)
- **Performance** -- pages load 27% faster on Windows, 11% faster on Mac (2025 improvements)
- **Custom emoji** -- upload custom emoji for workspace personalization
- **Mobile apps** -- full-featured iOS and Android apps with mobile AI, agents, and meeting notes

## 2. Obsidian

**Category:** Local-first markdown knowledge base and note-taking
**Revenue:** ~$2M | **Users:** 1.5M+ | **Bootstrapped** (no venture funding)
**Owner:** Dynalist Inc. (independent, bootstrapped)
**Platforms:** macOS, Windows, Linux, iOS, Android
**Pricing Model:** Free for all use; optional paid Sync and Publish services

### Editing & Content Creation

- **Plain-text Markdown** -- all notes are standard `.md` files stored locally on your device; no proprietary format or lock-in
- **Live Preview** -- WYSIWYG-like editing that renders markdown in real-time while you type
- **Source mode** -- raw markdown editing for power users
- **Reading mode** -- clean rendered view for reviewing content without editing
- **Rich text formatting** -- bold, italic, strikethrough, highlight, inline code, headings (H1-H6), blockquotes, horizontal rules
- **Checklists** -- task lists with checkbox syntax
- **Tables** -- markdown tables with column alignment
- **Code blocks** -- fenced code blocks with syntax highlighting for 200+ languages
- **Math/LaTeX** -- inline and block LaTeX equations via MathJax
- **Embeds** -- embed images, audio, video, and PDF files inline in notes
- **Image resizing** -- drag-to-resize images in Live Preview
- **Callouts** -- styled admonition/callout blocks with customizable types (tip, warning, note, etc.)
- **Footnotes** -- standard markdown footnote syntax
- **Mermaid diagrams** -- render Mermaid flowcharts, sequence diagrams, Gantt charts, etc. inline
- **Frontmatter/Properties** -- YAML frontmatter for structured metadata (tags, aliases, dates, custom fields)
- **Attachments** -- store images, PDFs, audio, and any file type in your vault alongside notes

### Linking & Knowledge Graph

- **Internal links** -- `[[wikilinks]]` syntax to connect notes bidirectionally
- **Backlinks** -- automatic panel showing all notes that link to the current note
- **Outgoing links** -- panel showing all links from the current note
- **Unlinked mentions** -- detect references to note titles even without explicit links
- **Graph view** -- interactive force-directed graph visualization of all notes and their connections; filter by tags, folders, or search; color-code by group
- **Local graph** -- focused graph showing only connections for the current note
- **Aliases** -- give notes multiple names for flexible linking
- **Block references** -- link to specific headings or blocks within notes using `^block-id` syntax

### Organization & Navigation

- **Vault-based** -- each vault is a folder of files on your filesystem; multiple vaults supported
- **File explorer** -- sidebar file tree for browsing vault contents
- **Quick switcher** -- fuzzy search to jump to any note instantly (Cmd/Ctrl+O)
- **Command palette** -- search and execute any command (Cmd/Ctrl+P)
- **Tags** -- `#tag` syntax with nested tag hierarchies (`#project/active`)
- **Tags view** -- dedicated panel listing all tags with counts
- **Starred/Bookmarks** -- pin notes, searches, and graph views for quick access
- **Workspaces** -- save and switch between layout configurations (open panes, sidebar state)
- **Search** -- full-text search across all notes with regex support; search by path, tag, property, or content
- **Folding** -- collapse headings and indented content
- **Outline** -- sidebar outline of all headings in the current note; drag to reorder sections
- **Tab support** -- open multiple notes in tabs; split panes horizontally or vertically
- **Stacked tabs** -- "Sliding panes" layout for lateral navigation through linked notes

### Canvas

- **Infinite canvas** -- freeform 2D workspace for visual note arrangement
- **Note cards** -- place existing notes as cards on the canvas
- **Text cards** -- create standalone text content directly on the canvas
- **Media cards** -- add images, PDFs, and web pages as cards
- **Connections** -- draw arrows/edges between cards to show relationships
- **Color coding** -- color-code cards and connections
- **Groups** -- visually group related cards together

### Templates & Automation

- **Templates (core)** -- insert predefined note templates with dynamic variables (date, time, title)
- **Daily notes** -- automatically create a new note for each day with customizable template and folder
- **Unique note creator** -- generate timestamped notes (Zettelkasten-style)
- **Slash commands** -- trigger commands inline while typing

### Core Plugins (30 built-in)

- Audio recorder, Backlinks, Bookmarks, Canvas, Command palette, Daily notes, File explorer, File recovery, Format converter, Graph view, Importer, Note composer (merge/split notes), Outgoing links, Outline, Page preview (hover preview), Properties view, Quick switcher, Random note, Search, Slash commands, Slides (presentation mode), Tags view, Templates, Unique note creator, Web viewer, Word count, Workspaces, and Bases (structured databases)

### Bases (Structured Databases) -- New in 2025

- **Bases** -- core plugin that turns collections of notes into structured databases
- **Views** -- table and other views for note collections
- **Properties-driven** -- uses note frontmatter properties as database columns
- **Filters and sorts** -- filter and sort notes by any property
- **Project management** -- track tasks, reading lists, or any structured collection

### Community Plugin Ecosystem

- **2,000+ community plugins** -- massive open-source plugin library extending functionality in every direction
- **Most popular plugins:** Dataview (database queries across vault), Templater (advanced dynamic templates), Excalidraw (freehand drawing/whiteboarding), Kanban (markdown kanban boards), Tasks (task management with due dates and recurring tasks), Calendar (calendar widget linked to daily notes), Advanced Tables (spreadsheet-like table editing), Git (version control backup), Copilot (AI assistant), Omnisearch (enhanced search), QuickAdd (quick capture and macro system), Style Settings (theme customization), Remotely Save (cloud backup to S3/WebDAV), Outliner (better outline/list editing), Homepage (custom startup page), Editing Toolbar (formatting toolbar), Iconize (custom icons), Minimal Theme Settings
- **CSS themes** -- 200+ community themes for complete visual customization
- **CSS snippets** -- apply custom CSS for fine-grained appearance tweaks

### CLI & Automation

- **Command line interface** -- control Obsidian from the terminal for scripting, automation, and integration with external tools
- **URI scheme** -- `obsidian://` protocol for deep-linking to specific notes, vaults, and actions
- **Siri/Shortcuts (iOS)** -- open and edit notes via Siri and Apple Shortcuts

### Import & Export

- **Importer plugin** -- migrate from Evernote, Notion, Apple Notes, Google Keep, OneNote, Roam Research, Bear, and HTML/Markdown files
- **Format converter** -- convert between markdown and other note formats
- **Export** -- notes are already plain markdown; copy/print/PDF export natively
- **No lock-in** -- all data is standard files on your device; switch tools anytime

### Security & Privacy

- **Local-first** -- all data stored on your device by default; no cloud dependency
- **No telemetry** -- zero analytics, tracking, or data collection
- **No account required** -- use the app without creating any account
- **End-to-end encryption (Sync)** -- optional cloud sync uses zero-knowledge E2E encryption

### Obsidian Sync (Paid Add-on)

- **Cross-device sync** -- sync vaults across macOS, Windows, Linux, iOS, and Android
- **End-to-end encryption** -- zero-knowledge encryption; even Obsidian cannot read your data
- **Version history** -- up to 12 months of version history for every note
- **Selective sync** -- choose which folders/files to sync
- **Multiple vaults** -- sync up to 10 vaults
- **5 GB storage** -- included per vault (expandable)

### Obsidian Publish (Paid Add-on)

- **Publish notes as website** -- turn selected notes into a public or password-protected website
- **Custom domain** -- use your own domain name
- **Navigation** -- sidebar navigation, search, and graph view on published site
- **Password protection** -- restrict access to published content
- **Custom CSS** -- style your published site with custom themes
- **Multiple sites** -- publish different vaults as separate sites
