# Competitor Features: Subscription Tracking and Alternative Budgeting Apps

Research compiled February 2026. Covers Rocket Money, EveryDollar, Goodbudget, and Bobby.

---

## 1. Rocket Money

**Category:** Subscription tracking + bill negotiation + budgeting hybrid
**Downloads/Users:** 4.1M premium members (acquired by Rocket Companies for $1.275B in 2024)
**Platforms:** iOS, Android, Web (premium only for web)
**Pricing:** Free tier + Premium ($7-$14/mo, pay-what-you-want model)

### 1.1 Subscription Detection and Management

| Feature | Description | Tier |
|---------|-------------|------|
| Automatic subscription detection | Scans linked bank/credit card transactions to identify recurring charges and subscriptions automatically | Free |
| Subscription dashboard | Centralized view of all detected subscriptions with amounts, billing dates, and trends over time | Free |
| Subscription cancellation (self-service) | Cancel subscriptions directly within the app with a few taps | Free |
| Concierge cancellation service | Rocket Money's team contacts providers on your behalf to cancel subscriptions, and attempts to recover refunds where possible | Premium |
| Upcoming bill timeline | Displays all upcoming bill due dates on a visual timeline so you never miss a payment | Free |
| Price change alerts | Notifies you when a subscription's price increases or a new recurring charge appears | Free |

### 1.2 Bill Negotiation

| Feature | Description | Tier |
|---------|-------------|------|
| Professional bill negotiation | Rocket Money's negotiators contact providers (phone, cable, internet, etc.) to lower your bills | Free (with success fee) |
| Pay-for-savings model | You only pay if the negotiation succeeds. Fee is 35-60% of the first year's savings (you choose the percentage within that range) | Free |
| Bill lowering concierge | Identifies which bills can be lowered and proactively recommends negotiation opportunities | Premium |

| Transaction tagging | Add custom tags to transactions for more granular tracking and filtering | Premium |
| Transaction splitting | Split a single transaction across multiple budget categories | Premium |
| Transaction notes | Add notes to individual transactions for context | Premium |
| Automation rules | Set rules to automatically categorize or tag transactions based on merchant or amount patterns | Premium |
| Manual transaction entry | Add cash or unlinked transactions manually | Premium |
| Net income tracking | Monthly snapshot showing total income, total spending, and what remains | Free |

### 1.4 Account Linking and Management

| Feature | Description | Tier |
|---------|-------------|------|
| Multi-account linking | Link checking accounts, savings accounts, credit cards, loans, and investment accounts from multiple institutions | Free |
| Automatic transaction sync | Transactions from linked accounts are automatically imported and categorized | Free |
| Real-time account syncing | On-demand sync to get the latest data from all linked financial institutions | Premium |
| Consolidated account overview | View all account balances in a single dashboard | Free |
| Account sharing | Share your Rocket Money account with family members or financial advisors for collaborative financial management | Premium |

### 1.5 Spending Analysis and Insights

| Feature | Description | Tier |
|---------|-------------|------|
| Spending insights | Personalized analysis of spending habits, identifying top spending categories and trends | Free |
| Spending reports | Generate detailed spending reports by category, merchant, and time period | Premium |
| Monthly/annual spending trends | View how spending changes month-over-month and year-over-year | Premium |
| Data export | Download transaction data as CSV for specific time periods, useful for tax prep or external analysis | Premium |

### 1.6 Alerts and Notifications

| Feature | Description | Tier |
|---------|-------------|------|

| High spending alerts | Alerts when credit card spending exceeds a set threshold | Free |

| New subscription alerts | Notifies you when a new recurring charge is detected | Free |

### 1.7 Savings

### 1.9 Platform and Customization

| Feature | Description | Tier |
|---------|-------------|------|
| iOS app | Full-featured mobile app for iPhone | Free |
| Android app | Full-featured mobile app for Android | Free |
| Web app (desktop) | Access Rocket Money through your browser on desktop/laptop | Premium |

## 2. EveryDollar

**Category:** Zero-based budgeting (Dave Ramsey ecosystem)
**Downloads/Users:** 12M+ downloads
**Platforms:** iOS, Android, Web (synced across all devices)
**Pricing:** Free tier + Premium (part of Ramsey+ membership, ~$49.99/quarter or ~$129.99/year after 14-day free trial)

### 2.1 Zero-Based Budgeting

| Feature | Description | Tier |
|---------|-------------|------|
| Zero-based budget creation | Assign every dollar of income to a specific category so income minus expenses equals zero | Free |
| Custom budget categories | Create unlimited spending categories tailored to your needs (wants, needs, savings, debt) | Free |
| Monthly budget planning | Plan your budget before the month begins, then track against it throughout the month | Free |
| Remaining balance tracking | See what is left in each category as you spend, with real-time updates | Free |
| Budget rollover | Carry unspent amounts or overages into next month's budget | Free |
| Budget templates | Start with suggested categories or build from scratch | Free |

### 2.2 Transaction Management

| Transaction splitting | Divide a single purchase across multiple budget categories (e.g., a Walmart trip split between groceries and household) | Premium |
| One-click tracking | Quickly confirm and categorize imported transactions with a single tap | Premium |
| CSV download | Export transaction data as CSV for external analysis or record-keeping | Premium |

### 2.3 Savings and Goals

| Feature | Description | Tier |
|---------|-------------|------|

| Goal tracking | Set target amounts and track progress toward each savings goal | Free |

### 2.4 Debt Payoff

| Debt progress tracking | See how much debt you have paid off and how much remains, with projected payoff dates | Premium |
| Baby Steps tracker | Follow Dave Ramsey's 7 Baby Steps framework with milestone tracking and recommended budget adjustments | Premium |

### 2.5 Financial Roadmap (Premium)

| Feature | Description | Tier |
|---------|-------------|------|
| Financial Roadmap | Personalized financial plan that projects milestones like becoming debt-free, based on your income and spending data | Premium |
| Net worth tracking | Monitor your total assets minus liabilities over time | Premium |
| Milestone projections | See projected dates for key financial milestones (debt freedom, fully funded emergency fund, etc.) | Premium |

### 2.6 Margin Finder

## 3. Goodbudget

**Category:** Pure envelope budgeting (no bank linking required on free tier)
**Downloads/Users:** 3M+ downloads
**Platforms:** iOS, Android, Web
**Pricing:** Free tier + Premium ($10/mo or $80/yr)

### 3.1 Envelope Budgeting System

| Feature | Description | Tier |
|---------|-------------|------|
| Digital envelope system | Divide your income into virtual envelopes for each spending category (groceries, rent, entertainment, etc.) | Free |
| Regular envelopes | Standard monthly spending envelopes for recurring categories | Free (10 max) / Premium (unlimited) |
| Annual envelopes | Set aside money each month for expenses that occur yearly (insurance, holidays, car registration) | Free (10 max) / Premium (unlimited) |
| Goal envelopes | Save toward specific financial goals like a vacation, new appliance, or emergency fund | Free (limited) / Premium (unlimited) |
| Envelope balance bars | Colorful visual bars on the home screen: green means money remains, red means you have gone over budget | Free |
| Automatic envelope fills | Set up recurring fills so envelopes automatically receive their allocated amount at the start of each budget period | Free |
| Flexible budget periods | Choose monthly, weekly, semi-monthly, or bi-weekly budget periods, starting on any day you choose, to match your pay schedule | Free |

### 3.2 Account and Transaction Management

| Feature | Description | Tier |
|---------|-------------|------|
| Manual transaction entry | Enter transactions by hand, assigning each to an envelope | Free |
| Bank account tracking | Track one bank account balance alongside envelope balances | Free (1 account) / Premium (unlimited) |
| Automatic bank sync | Connect bank accounts through Plaid to automatically import transactions | Premium |
| Smart envelope suggestions | When transactions are imported via bank sync, Goodbudget learns your patterns and suggests which envelope each transaction belongs to | Premium |
| Transaction matching | Manually match a manual entry with a bank-imported entry to create a single merged transaction, avoiding duplicates | Premium |
| Swipe to confirm | Swipe blue dots on the "Need to Confirm" screen to quickly confirm imported transactions instead of tapping multiple times | Premium (iPhone) |
| Scheduled transactions | Schedule future recurring transactions (e.g., mortgage, rent) with automatic posting on the due date | Free |
| Email notifications for scheduled transactions | Get email reminders a couple of days before a scheduled transaction posts | Free |
| CSV transaction export | Download all transactions as CSV for external use or record-keeping | Free |

### 3.3 Debt Management

| Feature | Description | Tier |
|---------|-------------|------|
| Debt accounts | Track debt balances (credit cards, loans, etc.) with dedicated debt accounts on the web | Free |
| Debt payoff tracking | Monitor progress on paying off each debt, with visual indicators of how much has been paid down | Free |
| Debt-free timeline | Project when you will become debt-free based on current payment amounts and schedule | Free |
| Goal saving while paying debt | Set aside funds for other financial goals simultaneously while making debt payments | Free |

### 3.4 Reports and Analysis

| Feature | Description | Tier |
|---------|-------------|------|
| Spending by envelope report | Pie chart breakdown showing how much was spent in each envelope/category | Free |
| Income vs. spending report | Compare total income to total spending over a selected time period | Free |
| Debt progress report | Visualize your debt payoff journey with trend charts | Free |
| Transaction history | View past transactions with 1 year of history on free, 7 years on premium | Free (1 yr) / Premium (7 yrs) |

### 3.5 Sharing and Multi-Device

| Feature | Description | Tier |
|---------|-------------|------|
| Multi-device sync | Budget syncs across iPhone, Android, and web in real time | Free (2 devices) / Premium (5 devices) |
| Household sharing | Share your budget with a partner, spouse, or family member so everyone sees the same envelope balances and transactions | Free (2 devices) / Premium (5 devices) |
| Cloud backup | All transaction data is backed up to the cloud and accessible from any device | Free |
| Web app | Full-featured web application at goodbudget.com for desktop budgeting | Free |

### 3.6 Platform and Design

| Feature | Description | Tier |
|---------|-------------|------|
| iOS app | iPhone app with envelope management, transaction entry, and reports | Free |
| Android app | Android app with full feature parity to iOS | Free |
| Web app | Browser-based app with additional features like debt accounts and expanded reports | Free |
| Ad-free experience | No advertisements on any tier | Free |
| Email support | Personal and friendly email support from the Goodbudget team | Premium |

### 3.7 Free vs. Premium Summary

| Capability | Free | Premium ($10/mo or $80/yr) |
|------------|------|---------------------------|
| Regular envelopes | 10 | Unlimited |
| Annual/goal envelopes | 10 | Unlimited |
| Accounts | 1 | Unlimited |
| Devices | 2 | 5 |
| Transaction history | 1 year | 7 years |
| Bank sync | No (manual or CSV upload) | Yes (automatic via Plaid) |
| Smart envelope suggestions | No | Yes |
| Email support | Community only | Personal email support |

## 4. Bobby

**Category:** Indie subscription tracker (manual entry, no bank linking)
**Downloads/Users:** Not publicly disclosed (featured in CNBC Select as "Best free service" for subscription trackers in 2026)
**Platforms:** iOS (primary), iPad, Mac (Apple Silicon), Apple Vision; unmaintained Android version exists
**Pricing:** Free (5 subscriptions) + one-time in-app purchases ($0.99-$2.99)

### 4.1 Subscription Tracking

| Feature | Description | Tier |
|---------|-------------|------|
| Manual subscription entry | Add subscriptions manually by choosing from hundreds of pre-loaded popular services or creating custom entries | Free (5 subs) / Paid (unlimited) |
| Pre-loaded subscription library | Hundreds of popular subscriptions (Netflix, Apple Music, Spotify, Dreamhost, Ulysses, Overcast, etc.) with their official icons and branding | Free |
| Custom subscriptions | Create entries for any subscription not in the library, with custom name, icon, and color | Free |
| Subscription list view | Clean, ordered list of all subscriptions showing name, price, next payment date, and custom icon | Free |
| Contract detail entry | Enter contract terms, billing frequency, and payment details for each subscription | Free |

### 4.2 Spending Overview

| Feature | Description | Tier |
|---------|-------------|------|
| Total cost calculation | View the total cost of all subscriptions in weekly, monthly, or yearly breakdowns by tapping the totals section | Free |
| Multi-currency totals | When subscriptions are in different currencies, Bobby shows a breakdown by currency and converts to your home currency automatically | Free |
| Category-based cost breakdown | See spending broken down by category (streaming, productivity, cloud storage, etc.) | Free |
| Quick payment calculator | Quickly calculate upcoming payment amounts for planning | Free |

### 4.3 Notifications and Reminders

| Feature | Description | Tier |
|---------|-------------|------|
| Bill due reminders | Set a reminder date so Bobby notifies you a configurable number of days or weeks before a subscription's payment date | Free |
| Per-subscription notification control | Disable notifications for specific subscriptions you do not need reminders about | Free |
| Consistent long-running notifications | Improved notification reliability for subscriptions you have tracked for a long time | Free |

### 4.4 Organization and Sorting

| Feature | Description | Tier |
|---------|-------------|------|
| Categories | Organize subscriptions into categories for filtered viewing and cost analysis | Free (limited) / Paid (unlimited) |
| Manual sorting | Drag subscriptions into your preferred order | Free |
| Automatic sorting | Sort by next due date, most recently added, etc. | Free |
| Alphabetical sorting | Sort A-Z or Z-A | Free |
| Price-based sorting | Sort by subscription cost (highest to lowest or reverse) | Free |
| Category filtering | Filter the subscription list to show only a specific category | Free |

### 4.5 Customization

| Feature | Description | Tier |
|---------|-------------|------|
| Dark mode | Toggle dark theme manually or set it on an automatic schedule | Paid (Visual Pack) |
| Light mode | Standard light theme | Free |
| Custom app icons | Choose from multiple alternative home screen icons for the Bobby app | Paid (Visual Pack) |
| Font options | Select from several font styles to personalize the app's appearance | Paid (Visual Pack) |
| Color picker | Choose custom colors for each subscription entry | Free |

| Color-coded subscriptions | Each subscription can have its own accent color for visual distinction | Free |

### 4.6 Currency Support

| Feature | Description | Tier |
|---------|-------------|------|
| Multiple currencies | Support for dozens of currencies including USD, EUR, GBP, PKR, MZN, BHD, MDL, and more | Free |
| Per-subscription currency | Set a different currency for each subscription (useful for international services) | Free |
| Daily exchange rate updates | Exchange rates are updated daily for accurate currency conversion | Free |

### 4.7 Privacy and Security

|---------|-------------|------|

| iCloud sync | Sync subscription data across devices via iCloud, with backup for data protection | Paid (All-in-One Pack) |

### 4.8 Platform Support

| Feature | Description | Tier |
|---------|-------------|------|

| iPad | Compatible with iPad for larger-screen viewing | Free |
