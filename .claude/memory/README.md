# Memory Runtime

This directory stores hook-managed session snapshots for MyLife.

- `runtime/` is generated by Claude Code hooks and is ignored by git.
- `precompact-latest.md` and `stop-latest.md` are rewritten on hook execution.
- `precompact-log.md` and `stop-log.md` append lightweight git-state snapshots for recovery.

Do not rely on these files as product documentation. They are operational breadcrumbs for long-running coding sessions.
